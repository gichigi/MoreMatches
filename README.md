# MoreMatches

Landing page for a dating profile tips service — real women in London give honest feedback on your profile.

Built for demand validation. Runs Meta ads to a single-screen CTA. No backend, no email capture yet.

## Stack

- Next.js 16
- Tailwind CSS v4
- shadcn/ui
- Vercel

## Getting started

```bash
pnpm install
pnpm dev
```

## Deploy

Deployed via Vercel. Push to `master` triggers a production deploy automatically.

> **Note:** This project was scaffolded from a [v0.dev](https://v0.dev) Vercel template. The original template included a full component library (shadcn/ui) and sections that are currently hidden — kept in the DOM for future use.
